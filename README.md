# Seki Motoki's PhD thesis figures

## Setup

### uv Installation
```bash
# Install uv
curl -LsSf https://astral.sh/uv/install.sh | sh

# Or via pip
pip install uv
```

### uv Update
```bash
# Update uv to latest version
uv self update
```

### Project Cache Update
```bash
# Clear and update project cache
uv cache clean
uv sync
```
